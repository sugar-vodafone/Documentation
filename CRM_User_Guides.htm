Vodafone CRM User Guides

Vodafone CRM Application Guide

The Vodafone CRM Application Guide introduces you to some basic Vodafone CRM concepts and helps you get familiar with the Vodafone CRM system. It describes how to access Vodafone CRM through a personal computer and a web browser. It also describes how to perform a broad range of customer relationship management tasks and related administration tasks.

Vodafone CRM Administration Guide

This Vodafone CRM Administration Guide describes options that enable Vodafone CRM administrators to configure the application to meet their organization's requirements.

Portal Guides

The Vodafone CRM Portal is a powerful tool that allows Vodafone CRM users to interact with customers and share information related to multiple modules (Bugs, Cases, Knowledge Base). This documentation will cover how to deploy, manage and use the portal.

    • Vodafone CRM Portal User Guide
    • Vodafone CRM Portal Deployment User Guide

Mobile Guides

VodafoneCRM Mobile for Android and iOS enables access to VodafoneCRM from users' Androids, iPhones or iPads to access data, manage daily activities, initiate phone calls, send SMS and emails, map record addresses and more while away from their computers. This documentation will cover how to install and use these mobile apps.

    • Vodafone CRM Mobile for Android User Guide
    • Vodafone CRM Mobile for iOS User Guide
